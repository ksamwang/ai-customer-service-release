# 智灵 AI 客服

智灵 AI 客服是面向电商商家的智能客服系统，以自研 LLM 为核心，围绕店铺接入、知识学习、自动接待、人工协同和订阅授权构建完整的本地化客户端能力。

系统通过本地客户端连接店铺数据与客服工作流，支持商家绑定店铺、沉淀商品与售后知识、配置接待策略，并由 AI 根据当前对话、知识证据和业务规则生成可控回复。

## 产品特点

- 自研 LLM 驱动，面向电商客服场景优化。
- 支持售前咨询、售后处理、商品问答、物流说明等高频客服场景。
- 店铺级知识体系，不同店铺可独立维护知识、策略和 AI Agent。
- AI 回复基于意图识别、证据召回、动作决策和安全约束，不是简单关键词匹配。
- 支持人工协同，AI 无法确定或存在风险时可进入人工待办。
- 基于账号订阅授权控制业务能力，按套餐管理有效期和可绑定店铺数量。
- 本地客户端运行，前端、桌面壳和本地服务协同，兼顾体验与稳定性。

## 当前支持平台

当前重点支持拼多多商家后台场景，已覆盖核心客服链路：

- 拼多多店铺绑定与授权
- 拼多多商家身份识别
- 拼多多客服会话接入
- 拼多多店铺级知识库与策略配置
- 拼多多自动回复与人工协同
- 拼多多多店铺授权数量控制

后续将逐步扩展更多电商平台，例如淘宝、天猫、抖店、快手小店、小红书店铺等。

## 发布与更新

该仓库作为智灵 AI 客服的公开发布通道，用于托管客户端更新元数据和发布资源。

更新入口：

- 最新版本信息：`latest.json`
- 前端资源：`frontend/`
- 本地服务：`local-service/`

客户端会通过该发布通道检查更新，并按需更新前端资源和本地服务组件。
